# EbookReader

Uses v0.3 of [EPUB.js](https://github.com/futurepress/epub.js) to parse and render epubs on Android using React Native.

Running the app
-------------------------

Install from NPM or Yarn

```bash
npm install
```
Then start the Android App

```bash
react-native run-android
```
